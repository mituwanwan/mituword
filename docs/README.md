## 迷途世界 (MituWorld) 项目简介

这是一个全新的项目阶段，实现以下核心功能：

- ✨ **双主题设计**  
  - 华丽宇宙（暗黑）：纯黑背景、星空、黑洞、星云辉光  
  - MC像素（白天）：我的世界风格、浮雕边框、像素字体、草地方块

- 🎵 **网易云音乐接入**  
  - 展示"我喜欢的"歌单  
  - 在线播放音频  
  - VIP歌曲降级处理（提示需客户端播放）

- 🔗 **好站分享**  
  - 分类展示推荐网站  
  - 后台管理（添加/编辑/删除）  
  - 前台搜索和筛选

- 📰 **每日日报**  
  - 自动生成日报（推荐音乐、好站、日记）  
  - 支持手动编辑和发布  
  - 历史日报归档

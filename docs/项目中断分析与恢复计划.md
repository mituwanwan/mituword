### 4.1 直接原因

**网络连接问题**：
- GitHub HTTPS连接失败：`Recv failure: 连接被对方重置`
- 代理已配置：`http://127.0.0.1:7897`，但连接仍失败

1. **代理配置可能不正确**：
   - 代理端口或协议可能不匹配

2. **GitHub访问限制**：

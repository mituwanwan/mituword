> 版本：v1.0  
> 日期：2026-06-04  
> 双主题：void（虚空/宇宙） / realm（领域/像素）

## 1. 字体系统

### 1.1 字体加载
```css
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&family=ZCOOL+KuaiLe&family=Inter:wght@400;500;600;700&display=swap');
```

### 1.2 字体映射
| 场景 | void（宇宙） | realm（像素） |
|------|-------------|---------------|
| 中文标题 | Inter + system-ui | ZCOOL KuaiLe |
| 中文正文 | Inter + system-ui | ZCOOL KuaiLe |
| 英文/数字 | Inter | Press Start 2P |
| 代码 | JetBrains Mono / Consolas | Press Start 2P |

### 1.3 字号规范
| 元素 | void | realm |
|------|------|-------|
| 页面大标题 | 48-64px, font-bold | 32-40px, font-pixel（像素字体不宜过大）|
| 区块标题 | 24-32px, font-semibold | 20-24px, font-pixel |
| 卡片标题 | 18-20px, font-medium | 16px, font-pixel |
| 正文 | 16px, leading-relaxed | 14-16px, font-pixel, leading-snug |
| 标签/辅助 | 12-14px | 12px, font-pixel |

```css
/* 标题渐变 */

.glow-purple {
  box-shadow: 0 0 20px rgba(109, 40, 217, 0.15), 0 0 40px rgba(109, 40, 217, 0.05);
}
```

## 3. 组件规范

### 3.1 卡片

**void（宇宙卡片）**
```tsx
<div className="
  bg-[#0a0a12]/80 
  backdrop-blur-sm 
  border border-[#1a1a2e] 
  rounded-xl 
  hover:border-[#6d28d9]/50 
  hover:shadow-[0_0_20px_rgba(109,40,217,0.15)]
  transition-all duration-300
">
```

**realm（像素卡片）**
```tsx
<div className="
  bg-[#C6C6C6] 
  border-4 
  border-t-white 
  border-l-white 
  border-r-[#555555] 
  border-b-[#555555]
  shadow-[4px_4px_0px_#000000]
">
```

### 3.2 按钮

**void（宇宙按钮）**
```tsx
<button className="
  bg-gradient-to-r from-[#6d28d9] to-[#db2777]
  text-white px-6 py-2 rounded-lg
  hover:shadow-[0_0_20px_rgba(109,40,217,0.4)]
  transition-all
">
```

**realm（像素按钮）**
```tsx
<button className="
  bg-[#7D7D7D] 
  border-4 
  border-t-white border-l-white 
  border-r-[#373737] border-b-[#373737]
  active:border-t-[#373737] active:border-l-[#373737]
  active:border-r-white active:border-b-white
  active:translate-y-[2px]
  px-4 py-2 font-pixel text-sm
">
```

### 3.3 输入框

**void**
```tsx
<input className="
  bg-[#0a0a12]/60 
  border border-[#1a1a2e] 
  rounded-lg px-4 py-2
  text-[#e0e7ff]
  focus:border-[#6d28d9] 
  focus:shadow-[0_0_10px_rgba(109,40,217,0.2)]
  outline-none transition-all
">
```

**realm**
```tsx
<input className="
  bg-white 
  border-4 
  border-t-[#373737] border-l-[#373737]
  border-r-white border-b-white
  px-3 py-2 font-pixel text-sm
  focus:border-[#5D8C22]
  outline-none
">
```

## 5. 布局规范

### 5.1 页面结构
```
┌─────────────────────────────┐
│         Header (fixed)       │
├─────────────────────────────┤
│                             │
│       WorldBackground        │
│       (fixed, z-[-1])        │
│                             │
├─────────────────────────────┤
│                             │
│         Page Content         │
│      (container mx-auto)     │
│                             │
├─────────────────────────────┤
│         Footer               │
├─────────────────────────────┤
│     Mini Music Player        │
│     (fixed bottom, optional) │
└─────────────────────────────┘
```

### 5.2 响应式断点
| 断点 | 宽度 | 说明 |
|------|------|------|
| sm | 640px | 手机横屏 |
| md | 768px | 平板 |
| lg | 1024px | 小桌面 |
| xl | 1280px | 标准桌面 |

### 5.3 容器宽度
- 默认：`max-w-7xl mx-auto px-4`
- 窄内容（日记阅读）：`max-w-3xl mx-auto px-4`

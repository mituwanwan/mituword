## 1. 项目概述

### 1.1 产品定位
迷途世界是一个融合个人作品集、技术博客、音乐分享和资源导航的个人网站。通过"双主题"设计（华丽宇宙暗黑主题 + 我的世界像素白天主题），打造独特的视觉记忆点。

### 1.2 目标用户
- 技术同行（查看项目、GitHub）
- 潜在雇主（了解技能、经历）
- 朋友/访客（听音乐、看好站分享）
- 回访用户（每日日报带来新鲜感）

### 1.3 核心价值
- **差异化体验**：宇宙×像素的双主题切换，全网罕见
- **内容聚合**：项目、日记、音乐、好站，一站式展示
- **日更动力**：每日日报机制，让访客有理由每天回来

## 3. 非功能需求

### 3.1 性能
- 首屏加载 < 2s（国内网络）
- Lighthouse 性能评分 > 80
- 图片使用 Next.js Image 优化

### 3.2 兼容性
- Chrome/Firefox/Safari/Edge 最新两版
- 移动端 iOS Safari + Android Chrome
- 不支持 IE

### 3.3 SEO
- 每个页面独立 title/description
- OG 标签完整
- 语义化 HTML 结构

### 3.4 安全
- 管理后台需要登录
- API 有 Rate Limiting
- 不暴露数据库凭据

## 4. 验收标准

### 4.1 主题切换验收
- [ ] 点击切换按钮，1s 内完成主题切换
- [ ] 刷新页面后保持上次选择的主题
- [ ] 所有组件正确响应主题变化

### 4.2 音乐板块验收
- [ ] 能正确获取"我喜欢的音乐"歌单
- [ ] 点击歌曲能播放音频
- [ ] 不可播放歌曲有明确提示+跳转链接
- [ ] 迷你播放器在所有页面可见

### 4.3 好站分享验收
- [ ] 后台能成功添加网站
- [ ] 前台能正确展示和筛选
- [ ] 点击跳转新窗口打开

### 4.4 日报验收
- [ ] 每天0点自动生成草稿
- [ ] 管理员可编辑并发布
- [ ] 首页正确展示当日内容
